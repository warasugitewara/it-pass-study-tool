## クイックスタート

### 1. アプリケーション起動
- デスクトップのアイコンをダブルクリック、または
- スタートメニューから「ITパスポート試験学習ツール」をクリック

### 2. メインウィンドウが表示される
```
┌─────────────────────────────────────────────┐
│  ITパスポート試験学習ツール                  │
├─────────────────────────────────────────────┤
│ [ファイル] [編集] [ツール] [ヘルプ]          │
├─────────────────────────────────────────────┤
│ ┌──────────────┐      ┌──────────────────┐ │
│ │  クイック    │      │  学習状況        │ │
│ │  スタート    │      │  ・合格率: 0%    │ │
│ │             │      │  ・解答数: 0     │ │
│ └──────────────┘      └──────────────────┘ │
│ ┌─────────────────────────────────────────┐ │
│ │ 出題方式選択                            │ │
│ │ ◎ ランダム出題                         │ │
│ │ ○ 年度別                              │ │
│ │ ○ 分野別                              │ │
│ │ ○ 復習モード                          │ │
│ └─────────────────────────────────────────┘ │
│ [テスト開始]                             │
└─────────────────────────────────────────────┘
```

### 3. テスト開始
1. 出題方式を選択（デフォルト: ランダム出題）
2. 「テスト開始」ボタンをクリック
3. 問題が表示されます

## UI 操作ガイド

### メニューバー

#### 📁 ファイルメニュー
```
ファイル
├─ 新規テスト (Ctrl+N)
├─ テストを開く (Ctrl+O)
├─ テストを保存 (Ctrl+S)
├─ ────────────
├─ エクスポート (Ctrl+E)
├─ インポート (Ctrl+I)
├─ ────────────
└─ 終了 (Alt+F4)
```

**各項目の説明:**
- **新規テスト**: 新しいテストセッションを開始
- **テストを開く**: 以前保存したテストを読み込む
- **テストを保存**: 現在のテストセッションを保存
- **エクスポート**: テスト結果をCSV/JSONでエクスポート
- **インポート**: 外部からの問題データをインポート
- **終了**: アプリケーションを閉じる

#### ✏️ 編集メニュー
```
編集
├─ 取り消す (Ctrl+Z)
├─ やり直す (Ctrl+Y)
├─ ────────────
├─ 切り取り (Ctrl+X)
├─ コピー (Ctrl+C)
├─ 貼り付け (Ctrl+V)
├─ ────────────
└─ すべて選択 (Ctrl+A)
```

#### 🔧 ツールメニュー
```
ツール
├─ 設定 (Ctrl+,)
├─ プラグイン管理
├─ ────────────
└─ キャッシュをクリア
```

**各項目の説明:**
- **設定**: ユーザー設定を変更
- **プラグイン管理**: 拡張機能の管理
- **キャッシュをクリア**: 一時ファイルを削除

#### ℹ️ ヘルプメニュー
```
ヘルプ
├─ オンラインヘルプ
├─ チュートリアル
├─ ────────────
├─ バージョン情報 (F1)
└─ フィードバック
```

## 各画面の説明

### メイン画面

#### 学習状況パネル（右上）
```
学習状況
正答率:     75%
解答数:     120問
進捗:       ████████░ 80%
```

表示される情報:
- **正答率**: これまでの解答で正解した割合
- **解答数**: 総解答問題数
- **進捗**: 学習全体の進捗度

#### 出題方式選択（中央）

**1. ランダム出題**
- 全問題からランダムに出題
- 問題数: カスタマイズ可能（デフォルト: 10問）
- 使用場面: 全体的な実力確認

**2. 年度別**
- 特定の年度の問題を出題
- 年度選択後に開始
- 使用場面: 過去問演習

**3. 分野別**
- 特定の分野（経営、IT など）の問題を出題
- 分野選択後に開始
- 使用場面: 分野集中学習

**4. 復習モード**
- 間違えた問題を出題
- 使用場面: 弱点克服

## 機能別チュートリアル

### チュートリアル1: 初回テスト実施

**目標**: ランダム出題で10問のテストを実施し、結果を確認する

**手順**:

1. **アプリケーション起動**
   - デスクトップのアイコンをダブルクリック

2. **メイン画面確認**
   - メイン画面が表示されることを確認

3. **出題方式確認**
   - 「ランダム出題」が選択されていることを確認

4. **テスト開始**
   - 「テスト開始」ボタンをクリック
   - テスト画面が表示される

5. **問題解答**
   - 各問題を読んで選択肢を選択
   - 矢印キーで移動、Space または Enter で決定
   - または、選択肢をマウスクリック

6. **テスト完了**
   - 最終問題で「次へ」をクリック
   - 結果画面が表示される

7. **結果確認**
   - 正答率と分野別成績を確認
   - 「詳細結果」をクリックで詳細表示

### チュートリアル2: 復習モード（間違えた問題の解き直し）

**目標**: 前回のテストで間違えた問題を復習する

**手順**:

1. **メイン画面を開く**
   - 「メニューへ」ボタンをクリック

2. **復習モード選択**
   - 「復習モード」ラジオボタンを選択

3. **テスト開始**
   - 「テスト開始」をクリック
   - 前回のテストで間違えた問題が出題される

4. **問題解答**
   - 同様の手順で問題を解答

5. **進捗確認**
   - テスト完了後、改善度を確認

### チュートリアル3: 分野別学習

**目標**: IT分野に特化した問題を10問解く

**手順**:

1. **分野別を選択**
   - 「分野別」ラジオボタンを選択

2. **分野を選択**
   - ダイアログから「IT」を選択
   - 問題数を入力（デフォルト: 10）

3. **テスト開始**
   - 「テスト開始」をクリック

4. **問題解答**
   - IT関連の問題が出題される

5. **結果確認**
   - テスト完了後、IT分野での成績を確認

### チュートリアル4: データのエクスポート

**目標**: テスト結果をCSVファイルとしてエクスポートする

**手順**:

1. **テスト完了**
   - テストを実施して結果画面に到達

2. **エクスポート選択**
   - メニュー → ファイル → エクスポート
   - または Ctrl+E を押す

3. **形式選択**
   - ダイアログで「CSV」を選択

4. **保存位置指定**
   - エクスポート先フォルダを選択
   - ファイル名を入力（デフォルト: test_results_YYYYMMDD.csv）

5. **完了**
   - 「保存」をクリック
   - 完了メッセージが表示される

## よくある質問（FAQ）

### Q1: アプリケーションが起動しない

**A**: 以下を確認してください：

1. **Windowsバージョン確認**
   ```
   Windows 10 以上が必要です
   設定 → システム → バージョン情報 で確認
   ```

2. **管理者権限での実行**
   ```
   アイコンを右クリック → 管理者として実行
   ```

3. **Visual C++ 再配布可能ファイルのインストール**
   ```
   Microsoft から ダウンロード・インストール
   ```

4. **アプリケーションの再インストール**
   ```
   アンインストール → インストーラーから再インストール
   ```

### Q2: テスト結果が保存されない

**A**: 以下を確認してください：

1. **自動保存設定確認**
   ```
   メニュー → ツール → 設定
   「テスト設定」で「自動保存」が有効になっているか確認
   ```

2. **データベースの再初期化**
   ```
   %APPDATA%\ITPassStudyTool\data.db を削除
   アプリケーションを再起動
   ```

3. **ディスク容量確認**
   ```
   Cドライブの空き容量が 100MB 以上あるか確認
   ```

### Q3: 問題データが古い／更新したい

**A**: 以下の手順でインポートしてください：

1. **新しい問題ファイルを用意**
   ```
   CSV形式: ID, 問題, 選択肢A, 選択肢B, 選択肢C, 選択肢D, 正解, 分野
   ```

2. **インポート実行**
   ```
   メニュー → ファイル → インポート
   または Ctrl+I を押す
   ```

3. **ファイル選択**
   ```
   ダイアログからCSVファイルを選択
   ```

4. **完了**
   ```
   インポート完了メッセージを確認
   ```

### Q4: 画面が小さすぎて見えない

**A**: 以下の設定を変更してください：

1. **コンパクト表示を無効化**
   ```
   メニュー → ツール → 設定
   「表示設定」で「コンパクト表示」をオフ
   ```

2. **フォントサイズ変更**
   ```
   設定画面でフォントサイズを大きくする
   ```

3. **全画面表示の解除**
   ```
   F11 キーで全画面表示を解除
   ```

### Q5: ダークモードを無効化したい

**A**:
```
メニュー → ツール → 設定
「表示設定」で「ダークモード」をオフ
「適用」をクリック
```

### Q6: 特定の分野の問題だけを削除したい

**A**: 現在、個別問題の削除機能はありません。

代案:
1. 管理パネルから問題一覧を表示
2. 削除対象をマーク
3. 「削除」ボタンで削除

または、データベースを初期化して再インポートしてください。

### Q7: 複数のユーザーで同じPCを使いたい

**A**: 各ユーザーアカウントで独立したデータベースが作成されます。

```
各ユーザー:
C:\Users\[ユーザー名]\AppData\Roaming\ITPassStudyTool\data.db
```

ユーザーを切り替えると、別のデータベースが使用されます。

### Q8: 正答率が0%と表示されている

**A**: これは正常な動作です。初回インストール時は、まだテストが実施されていないため、統計情報が存在しません。

テストを1回以上実施すると、正答率が計算・表示されます。

### Q9: テストを実施中に終了したい

**A**:
```
「テスト終了」ボタンをクリック
または
ウィンドウの × ボタンをクリック
```

確認ダイアログが表示されます。「はい」をクリックすると、テスト結果は保存されません。

### Q10: バックアップはどのように取ればよい？

**A**: 以下のフォルダをバックアップしてください：

```
Windows:
%APPDATA%\ITPassStudyTool\

手動バックアップ:
1. エクスポート機能でCSV形式で出力
2. 別のフォルダに保存
3. 新しいPCでインポート
```
