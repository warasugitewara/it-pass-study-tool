# 📍 MSI インストール後のアプリケーション配置場所

**バージョン**: v1.0.0  
**インストーラー**: ITPassStudyTool-1.0.0.msi  

---

## 🎯 インストール後の主な場所

### **1. プログラムファイル（メインアプリケーション）**

**インストール先**: `C:\Program Files\ITPassStudyTool\`

```
C:\Program Files\ITPassStudyTool\
├── Lib\                          # Python ライブラリ
├── Scripts\                       # Python スクリプト
├── tcl\                          # Tcl/Tk ライブラリ
└── python.exe                    # Python インタプリタ
```

**主要ファイル**:
- `python.exe` - Python インタプリタ
- `Lib/site-packages/` - インストール済みパッケージ
- `Lib/site-packages/src/` - アプリケーション本体

---

### **2. ショートカット（実行方法）**

#### **A) デスクトップ**
```
デスクトップ
└── ITパスポート試験学習ツール.lnk
```
**実行**: デスクトップの「ITパスポート試験学習ツール」アイコンをダブルクリック

#### **B) スタートメニュー**
```
スタートメニュー
└── すべてのプログラム
    └── ITPassStudyTool
        ├── ITパスポート試験学習ツール
        └── アンインストール
```
**実行**: スタートメニュー → すべてのプログラム → ITPassStudyTool → ITパスポート試験学習ツール

#### **C) クイックアクセス**
```
タスクバーの検索ボックス
→「ITパスポート」と入力
→ ITパスポート試験学習ツールが表示
```

---

### **3. データベースファイル（ユーザーデータ）**

**保存先**: `C:\Users\<ユーザー名>\AppData\Roaming\ITPassStudyTool\`

```
C:\Users\<ユーザー名>\AppData\Roaming\ITPassStudyTool\
├── app.db                    # SQLite データベース（問題・統計データ）
├── config.ini                # ユーザー設定ファイル
└── logs/                      # ログファイル
    ├── debug.log
    └── error.log
```

**説明**:
- `app.db`: 問題データ・正答率・学習時間などが保存される
- `config.ini`: ユーザーの設定（テーマ・出題数など）
- `logs/`: トラブルシューティング用ログ

---

### **4. レジストリ設定（Windows システム）**

**キー**: `HKEY_LOCAL_MACHINE\SOFTWARE\ITPassStudyTool`

```
HKEY_LOCAL_MACHINE\SOFTWARE\ITPassStudyTool
├── InstallPath          # インストール先パス
├── Version              # インストール済みバージョン
├── UninstallString      # アンインストール情報
└── DisplayIcon          # アイコン設定
```

**用途**: Windows の「プログラムの追加と削除」から削除情報を参照するために使用

## 🔧 アンインストール方法

### **方法 1: 「プログラムの追加と削除」から（推奨）**
```
1. Windows 設定を開く（Windows + I）
2. アプリ → インストール済みアプリ
3. 「ITパスポート試験学習ツール」を検索
4. 「アンインストール」をクリック
5. 確認画面で「削除」をクリック
```

### **方法 2: スタートメニューから**
```
1. スタートボタン → すべてのプログラム
2. ITPassStudyTool フォルダを開く
3. 「アンインストール」をクリック
4. 確認画面で「削除」をクリック
```

### **方法 3: MSI ファイルから**
```
1. ITPassStudyTool-1.0.0.msi をダブルクリック
2. インストーラー画面で「削除」を選択
3. 「削除」ボタンをクリック
```

**⚠️ 注意**: アンインストール時に `app.db` が削除される場合があります。
大切なデータは事前に **バックアップ** してください！

## 💾 バックアップ・復元

### **バックアップ方法**
```
1. C:\Users\<ユーザー名>\AppData\Roaming\ITPassStudyTool\app.db をコピー
2. 安全な場所（USB、クラウドストレージなど）に保存
```

### **復元方法**
```
1. app.db をバックアップファイルで上書き
2. アプリケーションを再起動
3. データが復元されます
```

## 🔍 トラブルシューティング

### **Q: アプリが起動しない**
```
A: 以下を確認してください
  1. デスクトップのショートカットが存在するか
  2. C:\Program Files\ITPassStudyTool\ が存在するか
  3. Python が正しくインストールされているか
  4. ログファイルを確認（C:\Users\...\AppData\Roaming\ITPassStudyTool\logs\）
```

### **Q: データが保存されていない**
```
A: 以下を確認してください
  1. app.db ファイルが存在するか
  2. C:\Users\<ユーザー名>\AppData\Roaming\ITPassStudyTool\ が存在するか
  3. ディスク容量が十分か
  4. ファイルの読み書き権限があるか
```

### **Q: デスクトップショートカットが見つからない**
```
A: 手動で作成してください
  1. スタートメニューから「ITパスポート試験学習ツール」を右クリック
  2. 「詳細」をクリック
  3. 「ファイルの場所を開く」をクリック
  4. ITパスポート試験学習ツール.lnk を右クリック → 送る → デスクトップ
```

### **Q: アンインストール後にファイルが残っている**
```
A: 以下を手動で削除してください
  1. C:\Program Files\ITPassStudyTool\（存在する場合）
  2. C:\Users\<ユーザー名>\AppData\Roaming\ITPassStudyTool\（データも削除）
```

## 📊 ストレージ使用量

| 項目 | サイズ | 説明 |
|------|--------|------|
| **インストール** | 150 MB | C:\Program Files\ |
| **ユーザーデータ初期** | 2-5 MB | app.db（初期状態） |
| **ログファイル** | 1-10 MB | 使用に応じて増加 |
| **合計** | 150-165 MB | 初期インストール時 |

**注**: ユーザーデータが増加するにつれて、`app.db` のサイズは増加します（クイズ実施数に応じて）

## 🎯 よくある質問（FAQ）

### **Q: インストール後、すぐに起動できますか？**
```
A: はい、デスクトップのショートカットをダブルクリックで即座に起動できます
```

### **Q: インストールに管理者権限は必要ですか？**
```
A: 必要です。インストール時に「管理者として実行」でセットアップしてください
```

### **Q: 複数のユーザーでインストールできますか？**
```
A: 各ユーザーが個別にインストール必要です
（各ユーザーの AppData に個別のデータベースが作成されます）
```

### **Q: インストール先を変更できますか？**
```
A: MSI インストーラーのカスタムインストールで変更可能です
（ただし Program Files 以下を推奨）
```

### **Q: オフラインで使用できますか？**
```
A: はい、インストール後はインターネット接続なしで使用可能です
（ただし、スクレイピング更新機能は接続が必要）
```
