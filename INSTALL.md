# ITパスポート試験学習ツール - インストールガイド

## 📋 システム要件

### 対応環境
- **OS**: Windows 10 以上（Windows 11 推奨）
- **プロセッサ**: Intel/AMD 1GHz 以上
- **メモリ**: 2GB 以上推奨（4GB 推奨）
- **ディスク容量**: 500MB 以上の空き容量
- **インターネット接続**: 初期化時に不要（インポート機能使用時に推奨）

### 動作確認済みOS
- Windows 10 (1909 以上)
- Windows 11 (全バージョン)

---

## 🔧 インストール方法

### 方法1: インストーラーを使用（推奨）

#### ステップ1: インストーラーをダウンロード
```
ITPassStudyTool-1.0.0-installer.exe
```

#### ステップ2: インストーラーを実行
1. ダウンロードしたファイルをダブルクリック
2. ユーザーアカウント制御（UAC）ダイアログが表示されたら「はい」をクリック

#### ステップ3: インストールウィザードに従う
1. **ウェルカムページ**: 「次へ」をクリック
2. **ライセンス契約**: 「同意する」を選択して「次へ」をクリック
3. **コンポーネント選択**:
   - ☑ ITパスポート試験学習ツール（必須）
   - ☑ スタートメニューショートカット（推奨）
   - ☑ デスクトップショートカット（オプション）
4. **インストール先ディレクトリ**: デフォルト（推奨）
   ```
   C:\Program Files\ITPassStudyTool\
   ```
   または任意のディレクトリを選択

#### ステップ4: インストール完了
1. インストール処理が自動で進行
2. 完了メッセージが表示されたら「完了」をクリック

#### ステップ5: アプリケーション起動
以下の方法でアプリを起動できます：
- デスクトップのショートカットをダブルクリック
- スタートメニューから「ITパスポート試験学習ツール」をクリック
- `C:\Program Files\ITPassStudyTool\it-pass-study-tool.exe` を直接実行

---

### 方法2: ソースコードからビルド

#### 前提条件
- Python 3.11 以上
- pip パッケージマネージャー
- NSISインストーラー（オプション）

#### ステップ1: リポジトリをクローン
```bash
git clone https://github.com/yourusername/it-pass-study-tool.git
cd it-pass-study-tool
```

#### ステップ2: 仮想環境を構築
```bash
python -m venv venv
venv\Scripts\activate
```

#### ステップ3: 依存ライブラリをインストール
```bash
pip install -r requirements.txt
```

#### ステップ4: アプリケーションを実行
```bash
python main.py
```

#### ステップ5: EXEファイルをビルド（オプション）
```bash
python build_exe.py
```

実行可能ファイルは `dist\it-pass-study-tool.exe` に生成されます。

#### ステップ6: NSISインストーラーを生成（オプション）
```bash
# NSIS がインストール済みの場合
cd path\to\nsis\bin
makensis.exe C:\path\to\it-pass-study-tool\setup.nsi
```

## ✅ インストール確認

インストール完了後、以下を確認してください：

### 1. アプリケーションの起動
```
アプリケーションが正常に起動し、メインウィンドウが表示されることを確認
```

### 2. メニュー動作確認
- ファイル → 新規テスト: 新しいテストセッションが開始される
- ツール → 設定: 設定画面が表示される
- ヘルプ → バージョン情報: バージョン情報が表示される

### 3. データベース確認
```
%APPDATA%\ITPassStudyTool\data.db が作成されていることを確認
```

## 🗑️ アンインストール方法

### 方法1: コントロールパネルから削除

1. Windows 設定を開く（`Win + I`）
2. 「アプリ」→「アプリと機能」
3. 「ITパスポート試験学習ツール」を検索
4. 「アンインストール」をクリック
5. 確認ダイアログで「はい」をクリック

### 方法2: スタートメニューから削除

1. スタートメニューを開く
2. 「ITPassStudyTool」フォルダを探す
3. 「アンインストール」のショートカットをダブルクリック
4. 確認ダイアログで「はい」をクリック

### 方法3: プログラムディレクトリから削除

1. `C:\Program Files\ITPassStudyTool\` を開く
2. `uninstall.exe` をダブルクリック
3. 確認ダイアログで「はい」をクリック

### アンインストール後の確認

以下を削除してください（自動削除されない場合）：

```
ユーザー設定データ:
%APPDATA%\ITPassStudyTool\

デスクトップショートカット（自動削除されない場合）:
C:\Users\[ユーザー名]\Desktop\ITパスポート試験学習ツール.lnk
```

## 🚨 トラブルシューティング

### インストール失敗時

#### 「管理者権限が必要です」エラー
**解決方法:**
1. インストーラーを右クリック
2. 「管理者として実行」を選択

#### 「Program Files へのアクセスが拒否されました」エラー
**解決方法:**
1. インストール先を `C:\Users\[ユーザー名]\AppData\Local\ITPassStudyTool` に変更
2. または、`C:\Program Files` の代わりに `C:\` 直下にインストール

### 起動エラー

#### 「python*.dll が見つかりません」エラー
**解決方法:**
```bash
# Microsoft Visual C++ 再配布可能ファイルをインストール
# https://support.microsoft.com/downloads/details.aspx?familyid=a7b7a05e-6de6-4d33-a490-61056a9080e7
```

#### 「設定フォルダへのアクセスが拒否されました」エラー
**解決方法:**
1. `%APPDATA%\ITPassStudyTool\` のフォルダを削除
2. アプリケーションを管理者として実行
3. フォルダが自動作成されます

### パフォーマンス関連

#### アプリケーションが遅い場合
**解決方法:**
1. メモリ使用量を確認（タスクマネージャー）
2. 不要なアプリケーションを終了
3. PCを再起動

#### データベースが破損した場合
**解決方法:**
```bash
# データベースを再初期化
# %APPDATA%\ITPassStudyTool\data.db を削除
# アプリケーションを再起動（自動的に再作成されます）
```
